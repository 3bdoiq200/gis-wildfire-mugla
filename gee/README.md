# GEE Scripts

Google Earth Engine JavaScript scripts for this project.

## Scripts (to be created)
- 01_flood_event_imagery.js — before/after flood satellite visualization
- 02_label_export.js — export flooded and non-flooded sample points
- 03_feature_layers.js — export all 15 flood susceptibility feature rasters
- 04_full_aoi_export.js — export full Karabük grid for prediction

## Instructions
1. Open code.earthengine.google.com
2. Create a new script for each file
3. Paste the contents
4. Define your AOI as Karabük Province boundary
5. Run and export to Google Drive
