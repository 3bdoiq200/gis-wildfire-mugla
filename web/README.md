# Web GIS Decision-Support System

Leaflet-based interactive flood susceptibility map for Karabük.

## Files (to be created)
- index.html — main application (self-contained, no server needed)
- data/ — GeoJSON layers for the map

## Features (planned)
- Interactive Leaflet map centered on Karabük
- Flood susceptibility layer with 5 risk classes
- Color legend (green to red)
- Layer toggles (susceptibility, rivers, admin boundary)
- Clickable popups with risk information
- Rainfall scenario slider (optional)

## How to run
Open index.html in any modern browser. No installation needed.
