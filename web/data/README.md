# web/data/

This folder holds the GeoJSON layer served to the Leaflet web app.

## Expected file

`flood_susceptibility.geojson` — generated by the last cell of
`colab/03_susceptibility_mapping.ipynb`.

If this file is missing, run Notebook 3 first and its final cell will
write the file here automatically.

## File size

Approximately 5–15 MB depending on point density. The notebook samples
every 3rd point from the full AOI grid (~40,000 points → ~13,000 in the
GeoJSON). If the browser is slow, re-run Notebook 3 with a larger step
(change `::3` to `::5` or `::10` in the last cell).

## Properties per feature

| Property | Type | Description |
|----------|------|-------------|
| FloodProb | float | Raw flood probability (0.0–1.0) |
| FloodPct  | float | Probability as percentage (0–100) |
| RiskClass | int   | Risk class 1–5 |
| RiskLabel | string | Very Low / Low / Medium / High / Very High |
